# 凉菜配方比例计算器

餐饮凉菜标准化配方工具。选择配方、输入主料重量，自动计算所有配料克数，生成操作步骤。

## 线上地址

https://peifang.ayakoai.com

## 功能

### 配方计算
- 选择配方 → 输入主料重量 → 自动计算所有配料克数
- 支持多种计算规则：百分比、比例、固定量、按份数
- 智能取整和范围限制
- 出水率预估和成品重量计算

### 步骤生成
- 自动生成操作步骤
- 变量高亮显示（食材名和克数）
- 一键打印备料清单

### 配方管理（manage.html）
- 新增 / 编辑 / 删除配方
- 配料拖拽排序
- 导入导出 JSON

### 生产记录（records.html）
- 每次计算自动记录
- 按日期查看生产历史
- 统计生产量

### 实用功能
- 分享配方链接
- 打印备料清单
- 数据本地缓存 + 服务端持久化

## 技术架构

- **前端：** 原生 HTML + CSS + JavaScript（零依赖，无构建步骤）
- **后端：** Node.js（Express）
- **数据：** JSON 文件存储（recipes.json / production-records.json）
- **部署：** 莱卡云服务器 + Nginx + Let's Encrypt SSL
- **设计：** 浅色清爽风格，绿色主色调（统一设计系统）
